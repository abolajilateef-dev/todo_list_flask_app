# Todo list App

A simple todo application built with Flask.

## Features

- Add tasks
- Mark tasks as complete
- Delete tasks
- Tasks persist in JSON file

  ## Built With

- Python
- Flask
- HTML/CSS

## How to Run

1. Run the app:
```
   python app.py
```
2. Open browser and go to `http://127.0.0.1:5000`

3. tick to mark as completed

4. clcikm X to remove task
